# Upraised-Assigment

The assignment is developed with the motto to search for active jobs by title/skill. 
The application is developed using NODE JS, EXPRESS, MYSQL, PUG and Python. 
The front end offers the  user to crawl over active jobs based on their skill or title input. 
The database consist of the title, job description, company, job created date and flag values to check for active/inactive jobs.
